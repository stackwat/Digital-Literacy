# Project Overview
This project focuses on developing digital literacy skills required in today’s digital world. It includes creating a professional online presence, understanding safe internet practices, learning coding basics, and spreading awareness about cybercrime. The project was completed through a series of tasks that involved using platforms like GitHub, LinkedIn, HackerRank, and Google Forms.
# Task Summaries
### Task 1: Digital Literacy Infographic
Created a one-page infographic using Canva that explains digital literacy, safe internet practices, and email etiquette in a simple and visually appealing manner.

### Task 2: Digital Portfolio
Developed a professional digital presence by creating profiles on GitHub, LinkedIn, and Kaggle, and setting up a GitHub profile README.

### Task 3: Coding & Collaboration Platforms
Solved a beginner-level problem on HackerRank and created a Digital Literacy Awareness Quiz using Google Forms. Responses were collected and viewed using Google Sheets.

### Task 4: Email Etiquette & Social Media Checklist
Drafted two professional emails and created a checklist of do’s and don’ts for responsible social media usage.

### Task 5: Cybercrime Awareness
Prepared a case study on UPI fraud and created a prevention checklist to promote safe online practices among students.
